This is the controller package.
This package contains the controller classes of the project.
The controller classes are responsible for handling the incoming HTTP requests and returning the HTTP responses.
The controller classes are annotated with @RestController annotation.
The controller classes are responsible for calling the service classes and returning the response to the client.
The controller classes are responsible for mapping the incoming HTTP requests to the appropriate service classes.
The controller classes are responsible for handling the exceptions and returning the appropriate error messages to the client.
The controller classes are responsible for validating the incoming HTTP requests and returning the appropriate error messages to the client.
The controller classes are responsible for converting the incoming HTTP requests to the appropriate service classes.
